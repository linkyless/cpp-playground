# mandelbrot

Complex numbers are essentially just 2D vectors with a specific multiplication rule. But if you take a point $c$ in the complex plane, start with $z_0 = 0$, and iterate a single quadratic function, the geometry breaks into infinite detail.

The function is simply:

$$z_{n+1} = z_n^2 + c$$

If the magnitude eventually escapes to infinity, the point is outside the Mandelbrot set. If it stays bounded forever, it is inside. That is the entire algorithm.

The boundary separating these two states is chaotic and self-similar at every scale. All that structural complexity is generated by a formula you can write on a napkin.
I built a visualizer for it.

![mandelbrot demo](mandelbrot.gif)

### how it works

Every pixel $(x, y)$ on the screen is mapped to a coordinate $c = x + yi$ in the complex plane. For each pixel, we compute the orbit.

If $|z_n| > 2$ at any step, the sequence is guaranteed to _diverge_. The loop breaks, and we record the exact number of iterations it took to escape. If it hits an arbitrary **iteration cap** without escaping, it is considered "trapped" inside the set.

This escape time drives the rendering. Points deep inside the set are drawn black. For the rest, the iteration count is mapped to an HSV color wheel to generate smooth gradients instead of harsh bands. So... the slower the escape, the richer the color.

### controls

| Input | Action |
| :--- | :--- |
| **Scroll Up** | Zoom in toward cursor |
| **Scroll Down** | Zoom out |

The zoom is centered on the cursor coordinates. You can navigate directly into any boundary region without losing focus.
