# cpp-playground

This is a scratchpad. There are no polished user-facing projects here. It is strictly a collection of some C++ stuff.

### compilation

Most files are standalone and do not require complex CMake configurations. They are written to be compiled directly via the terminal, usually with aggressive optimization flags to test maximum execution speed.

```bash
g++ filename.cpp -o output -O3 -Wall -Wextra
```
